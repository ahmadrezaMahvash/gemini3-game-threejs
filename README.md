# gemini3-game-threejs
A Game Build With Google Gemini

This is a Test For Measuring power Of Gemini 3 For Build Simple Game with Just One Prompt;

Prompt is Here :
Build a 3D Jumping over obstacles Game With JS,CSS,HTML and three.js Library All in One File. A Robot man She walks down the street.Various obstacles come in her way. User With " Space " , " Left" and "Right" Keys Can Can overcome obstacles. Load Robot Man With Three.js From This link " https://threejs.org/examples/#webgl_animation_skinning_blending".
